CSS
- 行内样式(内联样式）,直接在表现的style属性中书写
  &lt;p style="color: red; font-size: 60px; border:5px dashed purple"&gt;
- 页内样式，在本网页的style标签中书写
```
      <style>
        div{
            color:red;
            font-size: 30px;
            border: 4px solid yellow;
        }
        p{
            color: blue;
            font-size: 44px;
            background-color: yellowgreen;
        }
    </style>
```
